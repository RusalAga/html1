HEAD ELEMENTI :

head elementi metadata (məlumat haqqında məlumat) üçün konteynerdir və html teqi ilə body teqi arasında yerləşdirilir. Metadata HTML sənədi haqqında məlumatdır. Metadata göstərilmir. Metadata adətən sənədin başlığını, simvol dəstini, üslubları, skriptləri və digər meta məlumatı müəyyən edir. Aşağıdakı elementlər head elementinin içərisinə daxil ola bilər:

title
style
base
link
meta
script
noscript

BODY ELEMENTI :

body teqi sənədin gövdəsini müəyyən edir. body elementi HTML sənədinin bütün məzmununu, məsələn, başlıqlar, paraqraflar, şəkillər, hiperlinklər, cədvəllər, siyahılar və s. Qeyd: HTML sənədində yalnız bir body elementi ola bilər

H1-H6 ELEMENTLERI :

H1 - h6 teqləri HTML başlıqlarını müəyyən etmək üçün istifadə olunur. h1 ən vacib başlığı müəyyənləşdirir. h6 ən az əhəmiyyətli başlığı müəyyən edir.

P ELEMENTI :

p teqi paraqrafı müəyyən edir. Brauzerlər avtomatik olaraq hər p elementindən əvvəl və sonra bir boş sətir əlavə edir.

SPAN ELEMENTI :

span teqi mətnin bir hissəsini və ya sənədin bir hissəsini işarələmək üçün istifadə edilən daxili konteynerdir. span teqi asanlıqla CSS tərəfindən tərtib edilir və ya sinif və ya id atributundan istifadə edərək JavaScript ilə manipulyasiya edilir. span teqi div elementinə çox bənzəyir, lakin div blok səviyyəli elementdir və span daxili elementdir.

BR ELEMENTI :

br teqi tək sətir fasiləsi əlavə edir.
Br etiketi ünvanlar və ya şeirlər yazmaq üçün faydalıdır. br teqi boş teqdir, yəni onun son teqi yoxdur.
Qeyd: Paraqraflar arasında boşluq əlavə etmək üçün deyil, sətir sonlarını daxil etmək üçün br teqindən istifadə edin.

HR ELEMENTI :

hr teqi HTML səhifəsində tematik fasiləni təyin edir (məsələn, mövzunun yerdəyişməsi). Hr elementi ən çox HTML səhifəsində məzmunu ayırmaq (və ya dəyişikliyi müəyyən etmək) üçün istifadə olunan üfüqi qayda kimi göstərilir.

FORM ELEMENTI :

teqi istifadəçi daxiletməsi üçün HTML forması yaratmaq üçün istifadə olunur. elementi aşağıdakı forma elementlərindən birini və ya bir neçəsini ehtiva edə bilər:
input
textarea
button
select
option
optgroup
fieldset
label
output

DIV ELEMENTI :

div teqi HTML sənədində bölmə və ya bölməni müəyyən edir. div teqi HTML elementləri üçün konteyner kimi istifadə olunur - sonra CSS ilə tərtib edilir və ya JavaScript ilə manipulyasiya edilir. div teqi class və ya id atributundan istifadə etməklə asanlıqla tərtib edilir. İstənilən məzmun div teqinin içərisinə yerləşdirilə bilər.

SECTION ELEMENTI :

section teqi sənəddəki bölməni müəyyən edir.

TABLE>TH>TD>TR ELEMENTLERI :

table teqi HTML cədvəlini təyin edir. HTML cədvəli bir table elementindən və bir və ya daha çox tr, th və td elementlərindən ibarətdir. tr elementi cədvəl cərgəsini, th elementi cədvəl başlığını, td elementi isə cədvəl xanasını təyin edir. HTML cədvəlinə həmçinin caption, colgroup, thead, tfoot və tbody elementləri daxil ola bilər.

th> teqi HTML cədvəlində başlıq xanasını təyin edir. HTML cədvəlində iki növ hüceyrə var: Başlıq hüceyrələri - başlıq məlumatını ehtiva edir (th> elementi ilə yaradılmışdır) Məlumat hüceyrələri - verilənlərdən ibarətdir (td elementi ilə yaradılmışdır) th> elementlərində mətn qalın və defolt olaraq ortalanır. td elementlərindəki mətn normaldır və standart olaraq sola düzülür.

tr etiketi HTML cədvəlində bir sıra müəyyən edir. tr elementi bir və ya daha çox th> və ya td elementlərini ehtiva edir.

ARTICLE,ASIDE,HEADER,FOOTER ELEMENTLERI :

Məqalə teqi müstəqil, öz məzmunlu məzmunu müəyyənləşdirir. Məqalə özlüyündə məna kəsb etməlidir və onu saytın qalan hissəsindən müstəqil şəkildə yaymaq mümkün olmalıdır. Məqalə elementi üçün potensial mənbələr: Forum postu Bloq yazısı Xəbər hekayəsi Qeyd: Məqalə elementi brauzerdə xüsusi bir şey kimi göstərilmir. Başlıq elementi giriş məzmunu və ya naviqasiya keçidləri üçün konteyneri təmsil edir. header elementi adətən aşağıdakıları ehtiva edir: bir və ya bir neçə header elementi (h1 - h6) loqo və ya ikona müəlliflik məlumatları Qeyd: Bir HTML sənədində bir neçə header elementi ola bilər. Bununla belə, header altbilgi, ünvan və ya başqa başlıqda yerləşdirilə bilməz element.

aside> teqi, yerləşdirildiyi məzmundan başqa bəzi məzmunu müəyyənləşdirir. Kənara məzmun dolayısı ilə ətrafdakı məzmunla əlaqəli olmalıdır.

footer etiketi sənəd və ya bölmə üçün footer müəyyən edir. footer elementi adətən aşağıdakıları ehtiva edir: müəlliflik məlumatları müəllif hüquqları haqqında məlumat Əlaqə məlumatı sayt xəritəsi yuxarı keçidlərə qayıt əlaqədar sənədlər Bir sənəddə bir neçə altbilgi elementi ola bilər. bu linkden daha etrafli baxa bilersiniz

IMG ELEMENTI :

img teqi HTML səhifəsinə şəkil yerləşdirmək üçün istifadə olunur. Şəkillər texniki olaraq veb səhifəyə daxil edilmir; şəkillər internet səhifələrinə bağlanır. img teqi istinad edilən şəkil üçün saxlama yeri yaradır. img teqinin iki tələb olunan atributu var: src - Şəkilə gedən yolu müəyyən edir alt - Əgər hansısa səbəbdən şəkil göstərilə bilmirsə, şəkil üçün alternativ mətni təyin edir Qeyd: Həmişə şəklin enini və hündürlüyünü göstərin. Genişlik və hündürlük göstərilməyibsə, şəkil yüklənərkən səhifə yanıb-sönə bilər. İpucu: Şəkli başqa sənədlə əlaqələndirmək üçün sadəcə olaraq img teqini a teqinin içərisinə yerləşdirin
